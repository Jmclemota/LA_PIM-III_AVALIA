# LA_Dailys – Sprint 4

Esta pasta contém os registros das reuniões diárias (Daily Scrum) realizadas durante a Sprint 4 do projeto Avalia+.

---

##  Objetivo da Sprint

Finalizar a estrutura do projeto Avalia+, revisar a documentação técnica, concluir a modelagem do sistema e organizar os artefatos finais do PIM III.

---

##  Atividades da Sprint

Durante esta Sprint foram realizadas atividades relacionadas a:

- Revisão final da modelagem do sistema  
- Refinamento do diagrama de classes  
- Ajustes finais no banco de dados  
- Revisão e organização da documentação técnica  
- Refinamento dos protótipos e interfaces  
- Organização das Sprints, Reviews e Retrospectives  
- Estruturação final do repositório GitHub  
- Organização dos artefatos finais do PIM III  

---

##  Sobre as Dailys

As reuniões diárias tiveram como foco:

- Acompanhamento das entregas finais do projeto  
- Revisão da documentação técnica  
- Organização das tarefas finais da Sprint  
- Ajustes na modelagem e arquitetura do sistema  
- Refinamento dos protótipos e estrutura visual do projeto  
- Alinhamento final das responsabilidades da equipe  

As Dailys do projeto Avalia+ foram realizadas via Discord.
