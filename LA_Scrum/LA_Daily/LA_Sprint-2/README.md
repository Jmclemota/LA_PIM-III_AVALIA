# LA_Dailys – Sprint 2

Esta pasta contém os registros das reuniões diárias (Daily Scrum) realizadas durante a Sprint 2 do projeto Avalia+.

---

##  Objetivo da Sprint

Desenvolver a modelagem estrutural do sistema Avalia+, incluindo diagramas UML, fluxos de usuário, banco de dados e avanço da documentação técnica.

---

##  Atividades da Sprint

Durante esta Sprint foram realizadas atividades relacionadas a:

- Desenvolvimento dos diagramas UML  
- Estruturação do MER e DER  
- Organização dos fluxos de usuário  
- Desenvolvimento inicial do Dicionário de Dados
- Desenvolvimento inicial do Protótipo
- Refinamento dos requisitos do sistema  
- Evolução da documentação técnica  

---

##  Sobre as Dailys

As reuniões diárias tiveram como foco:

- Acompanhamento da modelagem do sistema  
- Organização das tarefas da Sprint  
- Revisão dos diagramas e banco de dados  
- Alinhamento das responsabilidades da equipe  
- Discussão sobre melhorias na arquitetura do sistema  

As Dailys do projeto Avalia+ foram realizadas via Discord.
