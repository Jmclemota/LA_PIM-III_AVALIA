# LA_UML – Projeto Avalia+

Esta pasta contém os diagramas UML do projeto Avalia+, representando a estrutura e o comportamento do sistema.

---

##  Sobre os Diagramas UML

| Diagrama | Finalidade |
|----------|-----------|
| **Casos de Uso** | Representa quais funcionalidades cada perfil de usuário pode acessar no sistema |
| **Classes** | Modela as entidades do sistema, seus atributos, métodos e relacionamentos |
| **Sequência** | Ilustra a ordem das interações entre objetos durante a execução de uma funcionalidade |
