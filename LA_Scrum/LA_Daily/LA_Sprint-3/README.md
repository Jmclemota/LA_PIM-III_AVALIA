##  Sobre as Dailys

A Daily Scrum é uma reunião curta realizada pela equipe para alinhar o andamento das atividades. Cada registro documenta:

- **Data e formato** da reunião
- **Participantes** presentes
- Atividades realizadas por cada membro desde a última daily
- Próximas atividades planejadas
- Impedimentos identificados
- Decisões e alinhamentos da reunião
- Data da próxima daily

As dailys do projeto Avalia+ são realizadas via **Discord**.
