##  Sobre a Metodologia

O projeto Avalia+ utiliza o **Scrum** como framework de desenvolvimento ágil, organizando as entregas em 4 sprints com duração de aproximadamente 2 semanas cada.

Ao final de cada sprint é realizada uma **Sprint Review** para apresentar o incremento entregue, e uma **Sprint Retrospective** para avaliar o processo da equipe.
