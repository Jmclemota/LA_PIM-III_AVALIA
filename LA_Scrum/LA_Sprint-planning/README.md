##  Sobre o Sprint Planning

O Sprint Planning é a cerimônia de abertura de cada sprint. Nela, a equipe define:

- O **objetivo da sprint**
- Os **itens do Product Backlog** que serão trabalhados
- A **distribuição de tarefas** entre os membros
- Os **resultados esperados** ao final do ciclo

Os documentos desta pasta registram as decisões tomadas em cada planning do projeto Avalia+.
