# LA_Sprint Review – Projeto Avalia+

## Sprint 3 – Desenvolvimento Inicial

**Data:** Abril de 2026  
**Duração da Sprint:** 2 semanas  

---

## Objetivo da Sprint

Iniciar o desenvolvimento estrutural do sistema Avalia+, avançando na modelagem backend em C#, organização do banco de dados e desenvolvimento das primeiras interfaces e protótipos do sistema.

---

## Entregas realizadas

Durante a Sprint 3, foram realizadas as seguintes atividades:

- Desenvolvimento inicial da modelagem backend em C#  
- Estruturação do diagrama de classes do sistema  
- Organização inicial das regras de negócio  
- Desenvolvimento da estrutura do banco de dados  
- Criação das primeiras interfaces e protótipos do sistema  
- Refinamento dos fluxos de navegação  
- Continuação da documentação técnica do projeto  
- Organização das Sprints, Dailys e Backlogs  

---

## Incremento do Produto

Ao final da Sprint 3, o projeto passou a contar com:

- Estrutura inicial da arquitetura backend definida  
- Avanço na modelagem orientada a objetos em C#  
- Banco de dados estruturado com base no DER  
- Evolução visual das interfaces do sistema  
- Refinamento dos fluxos de usuário  
- Melhor organização da documentação técnica do projeto
