# 📄 Docs – Projeto Avalia+

## 📋 Sobre a Documentação

A documentação do Avalia+ cobre todos os aspectos do sistema, desde a definição do problema até as regras que regem o funcionamento da plataforma. Ela serve tanto como referência técnica para o desenvolvimento quanto como entrega acadêmica do PIM III.
