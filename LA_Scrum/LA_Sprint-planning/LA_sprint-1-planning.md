# LA_Sprint Planning – Sprint 1

**Projeto:** Avalia+
**Data:** Março de 2026
**Duração da Sprint:** 2 semanas

---

# Objetivo da Sprint

Estabelecer a base inicial do projeto Avalia+, organizando o ambiente de desenvolvimento, iniciando a documentação do sistema e definindo os principais requisitos e estrutura do projeto.

---

# Itens selecionados do Product Backlog

Durante esta sprint foram priorizadas as atividades iniciais necessárias para a estruturação do projeto.

* Configuração do ambiente de desenvolvimento
* Criação do repositório do projeto no GitHub
* Levantamento inicial de requisitos do sistema
* Definição dos tipos de usuários do sistema
* Elaboração da introdução do projeto (documentação do PIM)
* Criação do Product Backlog
