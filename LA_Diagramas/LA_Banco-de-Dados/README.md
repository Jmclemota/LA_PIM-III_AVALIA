# 🗄️ Banco de Dados – Projeto Avalia+

Esta pasta contém a modelagem completa do banco de dados da plataforma Avalia+.

---

## 📋 Sobre a Modelagem

| Artefato | Descrição |
|----------|-----------|
| **MER** | Representa as entidades e seus relacionamentos em nível conceitual |
| **DER** | Representação gráfica das tabelas, chaves e relacionamentos do banco |
| **Dicionário de Dados** | Documenta cada tabela, campo, tipo de dado e restrições |

O banco de dados do Avalia+ utiliza um modelo **relacional**, armazenando informações de usuários, questões, simulados e resultados.
