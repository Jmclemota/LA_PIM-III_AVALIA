# LA_Dailys – Sprint 1

Esta pasta contém os registros das reuniões diárias (Daily Scrum) realizadas durante a Sprint 1 do projeto Avalia+.

---

##  Objetivo da Sprint

Estruturar a base inicial do projeto, organizar o repositório, iniciar a documentação técnica e definir os requisitos e perfis de usuários do sistema.

---

##  Atividades da Sprint

Durante esta Sprint foram desenvolvidas atividades relacionadas a:

- Organização inicial do projeto no GitHub  
- Levantamento dos requisitos do sistema  
- Estruturação da documentação técnica  
- Definição dos tipos de usuários  
- Criação do Product Backlog  
- Organização inicial das Sprints e Dailys  

---

##  Sobre as Dailys

As reuniões diárias tiveram como foco:

- Organização das tarefas da equipe  
- Definição das responsabilidades iniciais  
- Acompanhamento da documentação do projeto  
- Alinhamento das atividades da Sprint  
- Identificação de dificuldades iniciais do projeto  

As Dailys do projeto Avalia+ foram realizadas via Discord.
